# DigUp
[Dig] to the last page of the Internet, [Up]dated news and pages available, YaCy-based solution.<br>
_Note: Don't change project name, it's also GitHub Page URL._

# Goals
  - YaCy but federated-like on servers
  - Because P2P YaCy consumes too much CPU if runs locally on user's device

# Why YaCy?
  - P2P, popular and free
  - Easy to add to search index
